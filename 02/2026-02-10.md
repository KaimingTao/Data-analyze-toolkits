## Diff csv file

csv file mostly too long, too wide,
After making changes, how to know the difference?

- csv-diff
    - https://pypi.org/project/csv-diff/
- qsv
    - https://github.com/dathere/qsv
- xan
    - preview, filter, slice, aggregate, sort, join CSV files
    - https://github.com/medialab/xan
- miller
    - Miller is a command-line tool for querying, shaping, and reformatting data files in various formats, including CSV, TSV, JSON, and JSON Lines.
    - https://miller.readthedocs.io/en/6.16.0/
    - `mlr --csv diff old.csv new.csv`
- xsv
    - not maintained since Apr 24 2025
    - But I think I can still use it


## Git method

Tell Git how to diff CSV

Add this to .gitattributes:

*.csv diff=csv


And in .gitconfig:

[diff "csv"]
    textconv = xsv table


Now:

git diff data.csv
